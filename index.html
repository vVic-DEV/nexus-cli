Nexus CLI

Custom command-line toolset for developers

Install

npm install -g nexus-cli

After installation, you can run the nexus command globally.

Commands

nexus help

Show help for all commands.

nexus install <package>

Install a package using Nexus CLI.

nexus run <script>

Run a custom script.

nexus version

Show the current version of Nexus CLI.
